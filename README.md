# FITNESS HACKER - Health Guidance Website

## 📌 Project Overview

FITNESS HACKER is a health guidance website developed to promote healthy living and fitness awareness. The website provides exercise videos, wellness resources, and fitness guidance for different age groups, helping users maintain an active and healthy lifestyle.

---

## 🚀 Features

- User-friendly interface
- Health and fitness guidance
- Exercise and workout videos
- Age-specific fitness content
  - Kids
  - Teenagers
  - Adults
- Zumba workout sessions
- Image and video resources
- Responsive design
- Easy navigation

---
